# Joystick Arduino

Joystick modülü kullanan Arduino projesidir.

## Açıklama

Analog joystick kullanarak X ve Y eksenlerinden değer alma ve buton basışını algılayan Arduino projesidir. Robotic, oyun kontrolü veya hareket kontrolü uygulamalarında kullanılabilir.

## Kullanım

1. Joystick modülünü Arduino'ya bağla
2. Arduino IDE'yi aç
3. `JoystickArduino.ino` dosyasını yükle
4. Serial Monitor'u aç (9600 baud rate)
5. Joystick'i hareket ettir ve değerleri izle
6. Buton basışlarını test et

## Pinler

- VRx → A0
- VRy → A1
- SW → Pin 2
- +5V → 5V
- GND → GND
